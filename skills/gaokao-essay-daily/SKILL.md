# 高考作文模拟生成

本技能专注于高考作文的深度模拟与素材解析。它结合了百度教育"天工"素材库的高效应试风格，重点关注"科技与人文"、"个人与时代"等前沿命题趋势。

## 命令

### `/gaokao`

手动触发一次高考作文模拟生成。

**执行过程**：
1. **身份模拟**：以"优秀高中毕业生"的身份进行深度思辨与写作。
2. **趋势分析**：结合"天工"素材风格，重点关注科技向善、时代担当。
3. **自动同步**：文件将保存至 `documents/` 目录。
   - **GitHub 同步**：文件保存至 `documents/`。
   - **双重保障**：脚本会尝试自动运行 `git push`。
   - **本地获取**：您在本地电脑对应的 Git 目录执行 `git pull` 即可同步获取。

## 同步故障排查 (GitHub Token 配置)

由于云端环境没有您的登录信息，必须使用 **GitHub Personal Access Token (PAT)** 进行身份验证：

### 1. 获取 Token
- 前往 GitHub [Settings -> Developer settings -> Personal access tokens (classic)](https://github.com/settings/tokens)。
- 生成新 Token，勾选 `repo` 权限，并复制生成的字符串。

### 2. 配置方式（推荐：环境变量）
在 Zeabur 控制台的 **Environment Variables** 中添加：
- **`GITHUB_TOKEN`**: `您的Token字符串`
- **`OPENCLAW_GIT_SYNC`**: `true`

### 3. 配置方式（备选：手动设置 Remote）
在 Zeabur 的 **Command** 标签页运行：
```bash
git remote set-url origin https://<您的Token>@github.com/vgtmy/gaokao.git
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
```

## 生成规范

### 身份设定
你是一位资深的高考语文专家。请**以一名优秀高中毕业生的身份**，结合百度教育"天工"系列作文素材库的风格，完成一次高考作文模拟。

**核心任务**：
1. **身份设定**：你的思维、语气和文风应符合一名准大学生（高中毕业生）的特征，既有少年的朝气与担当，又不失深刻的思辨。
2. **内容质量**：参考天工素材库的"实用、系统、应试导向"。重点关注"科技向善与人文底线"或"个体成长与时代担当"。

### 输出结构规范（严格仅限以下三个板块）

1. ## 【高考作文模拟命题】
   - 给定一段具有时代感的背景材料，并明确写作要求。

2. ## 【破题思路】
   - **审题关键**：深入解析材料的时代背景、核心矛盾点，挖掘材料背后的多层内涵。
   - **立意建议**：提供3个不同层面的立意选择（如：从技术工具论、成长哲学、人文关怀等角度）。每个立意必须详细展开，说明该立意下的核心逻辑、论证切入点。
   - **架构与素材**：提供清晰的议论文结构蓝图，并推荐3-5条具有天工风格的高质量、非套路化素材。

3. ## 【范例】
   - 一篇800字左右的高分示范作文（模拟考场实战风格）。

### 禁止事项
- **严禁**输出"当前高考命题趋势分析"板块。
- **严禁**输出"写作技巧总结"或"高分要点"等专家点评板块。
- **严禁**输出除上述三个指定板块外的任何背景说明或引导语。

格式要求：Markdown。
